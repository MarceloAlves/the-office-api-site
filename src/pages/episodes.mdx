# Episodes

## The episode model

### Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the episode.
  </Property>
  <Property name="title" type="string">
    Title of the episode.
  </Property>
  <Property name="episode" type="string">
    The episode number
  </Property>
  <Property name="seriesEpisodeNumber" type="integer">
    The overall episode number in the series.
  </Property>
  <Property name="airDate" type="string">
    Original air date.
  </Property>
  <Property name="season" type="Season">
    The season information.
  </Property>
  <Property name="mainCharacters" type="Character">
    If included, the main characters for the episode.
  </Property>
  <Property name="supportingCharacters" type="Character">
    If included, the supporting characters for the episode.
  </Property>
  <Property name="recurringCharacters" type="Character">
    If included, the recurring characters for the episode.
  </Property>
</Properties>

## List all episodes {{ tag: 'GET', label: '/api/episodes' }}

<Row>
  <Col>

    Retrieves all episodes in the series and optionally includes characters.

    ### Optional attributes

    <Properties>
      <Property name="includeCharacters" type="boolean">
        Include characters in each episode.
      </Property>
      <Property name="limit" type="integer">
        Limit the number of episodes returned.
      </Property>
      <Property name="season" type="string">
        Limit episodes by season ID
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/api/episodes">

    ```bash {{ title: 'cURL' }}
    curl -G https://theofficeapi.dev/api/episodes \
      -d includeCharacters=true \
      -d season=640150ede4cd7dc24960d8fc \
      -d limit=10
    ```
